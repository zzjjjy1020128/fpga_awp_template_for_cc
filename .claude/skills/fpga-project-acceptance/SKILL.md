# 项目验收合同

使用 `ACCEPTANCE_CONTRACT_TEMPLATE.md` 模板创建项目验收合同。

验收合同定义：
- 每级验证的 pass/fail 标准
- 时序收敛目标（WNS/WHS）
- 资源占用上限（LUT/FF/BRAM/DSP/IOB 百分比）
- out-of-scope 范围

合同状态生命周期: unknown → draft → candidate → frozen → revised
