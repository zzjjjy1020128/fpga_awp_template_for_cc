# 项目章程

使用 `PROJECT_CHARTER_TEMPLATE.md` 模板创建项目章程。

项目章程定义：
- 项目范围和目标
- 关键干系人
- 技术和非技术约束
- 高层级时间线和里程碑
- 验收标准概要

## 反模式

### ❌ "小项目不需要章程"
```
即使单人/单模块项目，章程明确 scope 边界和验收标准。
没有章程的默认行为 = scope creep + 永无止境的"再完善一下"。
```
